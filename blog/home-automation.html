How I Built A Home Automation System

Posted on November 8, 2025

This is my first project using an arduino and esp8266 mcu. I designed it such that users are able to use/control their home appliances using a mobile device.

What is Home Automation

Home automation is building automation for a home, called a Smart Home or Smart House. A home automation system will monitor and/or control home appliances like Lights, Fans, Windows, etc. These home appliances are connected with a remote device using Internet of Things("IoT") for controlling/monitoring the home appliances.

Components Used

A.Software Components: • Arduino IDE • ESP8266 Library • Blynk Libraries • Blynk App B. Hardware • Node MCU • Rellays • Appliances (LED, Fan, Window) • Battery
