How I Built A Sonar Lander

Posted on November 8, 2025

This is my second project using an arduino due mcu. I designed it such that the landing gear is automatically deployed upon reaching the threshold height .

Automatic Landing Gear

we developed a model planetary lander using an Arduino microcontroller, an ultrasonic distance sensor, and servo motors to deploy landing gear automatically. By simulating planetary descent,this project explores the principles of altitude detection and mechanical deployment for safe landings, demonstrating the engineering design process and the potential to avoid crash landings

Components Used

• Arduino-compatible microcontroller board • Ultrasonic distance sensor • SG90 micro servo motors (4) • Breadboard and jumper wires • 9V battery with snap connector
